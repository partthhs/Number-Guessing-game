8NUMBER GUESSING GAME<br>

•Simple console-based Number Guessing Game built using Java. The program generates a random number, and the player has to guess it within a limited number of attempts.<br>

📌 Features<br>
1️⃣Random number generation<br>
2️⃣User input handling<br>
3️⃣Feedback on guesses (Too High / Too Low)<br>
4️⃣Limited number of attempts<br>
5️⃣Simple and beginner-friendly logic<br>
